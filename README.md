## Introduction
### Biological Background
  A significant aspect of immunology is determining how viruses infect different cell types. Since different cells have varying levels of defenses to viral infections and viruses are constantly mutating and changing how they infect cells, there is constantly new information to be learned in this field (Sutherland et al., 2019). Even within organisms the arms race between infection and immunity grows increasingly complex as different tissue types have different capabilities (Kazer et al., 2024). When studying how different tissue types are impacted by infection, scientists often choose to study mice as this reduces the ethical concerns of using human subjects while also remaining highly replicatable and similar to human (Manikkaraja et al., 2020). When determining how the genetic structure of cells are impacted, single-cell RNA sequencing becomes a useful tool (Kazer et al., 2024). In this assignment I will be using the data collected from Kazer et al. to take a bioinformatics approach and determine differences between tissue types when infected nasally with influenza (Kazer et al., 2024).
### Methods Justification
  For this analysis I will be using Seurat (v. 5.4.0) in RStudio (v. 4.5.0). Seurat is a broadly-used package for single-cell RNA sequence analysis (Heumos et al., 2023). While Seurat handles the large dataset well, it does have some pitfalls, including a high false positive rate when performing differential expression analyses for which I have opted to use DESeq2 (v. 1.50.2) integrated into Seurat (Panahi et al., 2025). 
## Methods
The data from Kazer et al. had been collected as a Seurat object. From there the data was normalized and scaled using default parameters. Then the data was clustered with a resolution of 0.30. A UMAP plot was then generated using ggplot2 (v. 4.0.2) which had 31 groups numbered 0-30. The differential expression analysis was focused on cluster 3 and compared the LNG (lateral nasal gland) cell type to the RM (respiratory mucosa) and OM (olfactory mucosa). This analysis was run using DESeq2 outside of Seurat as this option was more capable of running the analysis with the labelling of the data as it was. A volcano plot was generated using ggplot2 to highlight the differential expression of the cells. Following this, feature plots were made that showcased the activity of Ifit1, Obp1a, Fosl2, and Gnb1 throughout the cells. These plots were generated using ggplot2. A GSEA was then performed to give biological context to the differential expression analysis and demonstrate what processes are up regulated or downregulated in the LNG cells in comparison to the OM and RM cells. To visualize this a dotplot was generated using ggplot2.
## Results
  The generated UMAP demonstrates 31 different cell clusters and if the UMAP is made to highlight the different cell types we see how these cells are split (Figure 1). Looking specifically at group 3, all three cells types are involved but the differential expression analysis compares the LNG cells with OM and RM cells demonstrates that there are clear differences in the genetic expression of cells in this group (Figure 2). Performing a GSEA on this expression analysis, it is determined that most processes are downregulated in LNG cells in comparison to the others, with the exception of pheromone response (Figure 3). 
  When examining individual proteins involved in this sequence, four in particular stood out. Obp1a is likely a protein strongly expressed in LNG cells and it being primarily expressed in subgroup 18 indicates that this subgroup is likely responsible for pheromone/odor reception (Figure 4). In addition, Ifit1 and Fosl2 were found to be primarily expressed in RM cells. Gnb1 had by far the most significant expression and was found in RM and OM cells.\

## Discussion
  The findings of this assignment demonstrate that the LNG cells in the mouse respiratory system are likely more involved with pheromone and odor detection rather than immunity from infections. The Obp1a, Ifit1, Fosl2, and Gnb1 genes all encode for different processes within the nasal system of the mouse (Johnson et al., 2018; Manikkaraja et al., 2020; Shetty et al., 2022; Zhang et al., 2023). Future contributions could consider how expression might differ in response to different disease types.
